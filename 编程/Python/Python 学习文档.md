# Python学习文档

## 一、Python 变量核心要点

### **变量创建与特性**



- 无需提前声明，**首次赋值时自动创建**
- 动态类型：无需指定类型，赋值后可随时更改数据类型
- 字符串变量可用单引号`' '`或双引号`" "`声明，效果一致



### **变量命名规则（强制）**



- 只能以**字母或下划线**开头，不能以数字开头
- 仅允许包含字母、数字和下划线（A-z、0-9、_）
- **严格区分大小写**（`age`、`Age`、`AGE`是三个不同变量）



### **多变量赋值**



- 一行赋多个不同值：`x, y, z = "Orange", "Banana", "Cherry"`
- 一行赋多个相同值：`x = y = z = "Orange"`



### **变量输出与运算**



- 使用`print()`语句输出变量
- `+`号：字符串间用于拼接，数字间用于加法运算
- **禁止直接拼接字符串和数字**，会触发语法错误



### **全局变量与`global`关键字**



- 函数外部创建的是**全局变量**，可被函数内外访问
- 函数内部同名变量默认是**局部变量**，仅在函数内生效，不影响全局变量
- `global`关键字作用：
  - 在函数内部创建全局变量
  - 在函数内部修改已存在的全局变量的值

## 二、Python 语法核心要点

### **语法执行方式**



- 命令行直接执行：在 Python 交互环境（>>>）中输入代码立即运行
- 文件执行：创建`.py`后缀的脚本文件，通过命令行`python 文件名.py`运行



### **缩进（Python 特有且强制）**



- 作用：用代码行开头的空格**标识代码块**，而非仅用于可读性
- 规则：至少 1 个空格；**同一代码块必须使用相同数量的空格**，否则触发语法错误
- 反例：省略缩进、同一代码块空格数不一致都会报错



### **变量**



- 无需提前声明，**赋值时自动创建**
- 示例：`x=5`（数值型）、`y="Hello, World!"`（字符串型）



### **注释**



- 单行注释以`#`开头，Python 会忽略`#`后的整行内容
- 作用：对代码进行说明，不影响程序运行



## 三、Python 数据类型

### **内置数据类型（按类别）**



- 文本：**str**
- 数值：**int、float、complex**
- 序列：**list、tuple、range**
- 映射：**dict**
- 集合：**set、frozenset**
- 布尔：**bool**
- 二进制：**bytes、bytearray、memoryview**

#### 1、 文本类型：`str`（字符串）

- **定义**：由字符组成的不可变序列，用于表示文本。

- **特点**：用单引号 `'`、双引号 `"` 或三引号 `'''`/`"""` 包裹；支持索引、切片、拼接等操作。

- **示例**：

  python

  

  ```
  s1 = 'Hello'
  s2 = "Python"
  s3 = '''多行

  ```

  

#### 2. 数值类型

##### （1）`int`（整数）

- **定义**：表示没有小数部分的数字，可正可负。
- **示例**：`100`、`-42`、`0`。

##### （2）`float`（浮点数）

- **定义**：表示带有小数部分的数字，支持科学计数法。
- **示例**：`3.14`、`-0.5`、`2.0e3`（即 2000）。

##### （3）`complex`（复数）

- **定义**：由实部和虚部组成，虚部用 `j` 表示。
- **示例**：`1+2j`、`-3j`、`complex(4,5)`（即 4+5j）。

##### 类型转换

您可以使用 `int()`、`float()` 和 `complex()` 方法从一种类型转换为另一种类型：

##### **随机数**

- Python 无直接 `random()` 函数
- 需导入 `random` 模块，用 `random.randrange(起始, 结束)` 生成随机数

##### 字符串字面量

python 中的字符串字面量由单引号或双引号括起。

`'hello'` 等同于 `"hello"`。

#### 3. 序列类型

##### （1）`list`（列表）

- **定义**：可变的有序元素序列，元素可重复、类型可不同。
- **特点**：用方括号 `[]` 包裹；支持增删改查。
- **示例**：`[1, 'a', 3.14]`、`[]`（空列表）。

##### （2）`tuple`（元组）

- **定义**：不可变的有序元素序列，类似列表但不能修改。
- **特点**：用圆括号 `()` 包裹；若只有一个元素，需加逗号 `(1,)`。
- **示例**：`(1, 'a', 3.14)`、`()`（空元组）。

##### （3）`range`（范围）

- **定义**：生成一个整数序列，常用于循环。
- **特点**：不可变，占用内存小（仅存储起始、结束和步长）。
- **示例**：`range(5)`（0-4）、`range(1,10,2)`（1,3,5,7,9）。

#### 4. 映射类型：`dict`（字典）

- **定义**：无序的键值对（`key-value`）集合，键必须唯一且不可变。
- **特点**：用花括号 `{}` 包裹，通过键访问值。
- **示例**：`{'name': 'Alice', 'age': 20}`、`{}`（空字典）。

#### 5. 集合类型

##### （1）`set`（集合）

- **定义**：无序、不重复的元素集合。
- **特点**：用花括号 `{}` 或 `set()` 创建；支持交、并、差等集合运算。
- **示例**：`{1,2,3}`、`set([1,2,2,3])`（去重后为 {1,2,3}）。

##### （2）`frozenset`（不可变集合）

- **定义**：不可变的集合，类似 `set` 但不能增删元素。
- **特点**：可作为字典的键或其他集合的元素。
- **示例**：`frozenset({1,2,3})`。

#### Python 列表核心总结

1. **列表是什么**

   

   - 有序、可修改、允许重复元素，用 **方括号 []** 表示。

   

2. **访问元素**

   

   - 正索引：从 0 开始（list [0] 是第一个）
   - 负索引：-1 最后一个，-2 倒数第二个
   - 范围切片：list [2:5]（含左不含右）

   

3. **修改元素**

   

   - 直接赋值：list [index] = 新值

   

4. **常用操作**

   

   - 遍历：for x in list:
   - 判断存在：if 值 in list
   - 长度：len (list)

   

5. **添加元素**

   

   - 末尾加：append (值)
   - 指定位置插：insert (索引，值)
   - 批量追加：extend (另一个列表)

   

6. **删除元素**

   

   - 按值删：remove (值)
   - 按索引删：pop (索引) / 无索引删最后一个
   - 删除整个列表：del list
   - 清空内容：clear ()

   

7. **复制列表**

   

   - 不能直接 list2 = list1（会关联）
   - 正确复制：copy () 或 list ()

   

8. **合并列表**

   

   - 用 + 号 或 extend ()

   

9. **创建列表**

   

   - 直接：[]
   - 构造函数：list (())

   

10. **常用方法**

    

    - append、insert、extend、remove、pop、clear、copy、count、index、reverse、sort

#### 6. 布尔类型：`bool`

- **定义**：只有两个值：`True`（真）和 `False`（假），用于逻辑判断。

- **示例**：`1 > 2` 结果为 `False`，`3 == 3` 结果为 `True`。

- ## 大多数值都为 True

  如果有某种内容，则几乎所有值都将评估为 True。

  除空字符串外，任何字符串均为 True。

  除 0 外，任何数字均为 True。

  除空列表外，任何列表、元组、集合和字典均为 True。

  ### 某些值为 False

  实际上，除空值（例如 ()、[]、{}、""、数字 0 和值 None）外，没有多少值会被评估为 False。当然，值 False 的计算结果为 False。

#### 7. 二进制类型

##### （1）`bytes`（字节）

- **定义**：不可变的字节序列，每个元素是 0-255 的整数。
- **特点**：用 `b` 前缀表示。
- **示例**：`b'hello'`、`bytes([104,101,108,108,111])`。

##### （2）`bytearray`（字节数组）

- **定义**：可变的字节序列，类似 `bytes` 但可修改。
- **示例**：`bytearray(b'hello')`、`bytearray([104,101,108,108,111])`。

##### （3）`memoryview`（内存视图）

- **定义**：访问其他二进制对象的内存而不复制数据，用于高效处理大文件。
- **示例**：`memoryview(b'hello')`。

### **获取数据类型**

用 `type(变量)` 查看数据类型，例：`x=10; print(type(x))`

### **设置数据类型**

- 自动设置：**赋值时自动确定类型**
- 手动指定：用**构造函数**强制设定，例：`x=str("Hello")`、`x=int(29)`

## 四、Python 运算符

Python 运算符共分为七类，用于对变量和数值执行运算、赋值、比较、判断等操作。

- **算术运算符**：用于数学计算，包含 + - * / % ** //，分别对应加、减、乘、除、取模、幂运算、取整除法。
- **赋值运算符**：用于给变量赋值，包含 =、+=、-=、*=、/=、%=、//=、**= 等，可简化运算赋值写法。
- **比较运算符**：用于比较两个值，返回布尔结果，包含 ==、!=、>、<、>=、<=。
- **逻辑运算符**：用于组合条件，and 表示全真为真，or 表示一真为真，not 表示取反结果。
- **身份运算符**：用于判断是否为同一对象（同内存地址），包含 is、is not。
- **成员运算符**：用于判断值是否在序列中，包含 in、not in。
- **位运算符**：对二进制数进行位操作，包含 &、|、^、~、<<、>>。



## 五、Python 条件判断（If...Else）

Python 通过 `if/elif/else` 实现条件分支逻辑，**核心依赖缩进定义代码块**（无花括号），所有条件最终返回布尔值 `True`/`False`。

### 一、基础比较运算符（返回布尔值）

用于两个值的大小 / 相等判断，是条件语句的基础：

表格







| 运算符 | 含义     | 示例     |
| :----- | :------- | :------- |
| `==`   | 等于     | `a == b` |
| `!=`   | 不等于   | `a != b` |
| `<`    | 小于     | `a < b`  |
| `>`    | 大于     | `a > b`  |
| `<=`   | 小于等于 | `a <= b` |
| `>=`   | 大于等于 | `a >= b` |

### 二、核心条件分支结构

### 1. 单分支（if）

仅当条件为 `True` 时执行代码块

python



运行









```
# 语法
if 条件:
    条件成立时执行的代码（必须缩进）

# 示例
a = 66
b = 200
if b > a:
    print("b is greater than a")
```

### 2. 双分支（if...else）

条件成立走 `if` 块，不成立走 `else` 块

python



运行









```
# 语法
if 条件:
    条件成立执行
else:
    条件不成立执行

# 示例
a = 200
b = 66
if b > a:
    print("b is greater than a")
else:
    print("b is not greater than a")
```

### 3. 多分支（if...elif...else）

按顺序判断条件，**第一个成立的分支执行后，后续分支全部跳过**；`else` 是兜底分支（可选）

python



运行









```
# 语法
if 条件1:
    条件1成立执行
elif 条件2:
    条件1不成立、条件2成立执行
elif 条件3:
    前两个都不成立、条件3成立执行
else:
    所有条件都不成立执行

# 示例
a = 66
b = 66
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")
```

### ⚠️ 关键注意：缩进规则

- Python 强制用**4 个空格**缩进定义代码块（推荐），不能用 Tab 或 Tab + 空格混合
- 同一代码块的缩进必须完全一致，否则会报语法错误
- 没有缩进的代码不属于条件分支，会无条件执行

### 四、多条件组合（逻辑运算符）

用于同时判断多个条件

表格







| 运算符 | 逻辑规则                               | 示例              |

### 五、嵌套 If 语句

### 六、pass 语句

- 作用：**空代码占位符**
- 场景：if 语句块暂时没有要执行的代码，但不能留空（留空会报语法错误）

# 六、Python 类与对象 

1. **基础定义**

   

   Python 是面向对象语言，

   类

   是创建对象的蓝图 / 构造器，

   对象

   拥有属性（变量）和方法（函数）。

2. **创建语法**

- 用 `class` 关键字定义类；
- 用 `类名()` 语法创建类的实例对象。

1. **`__init__()` 初始化函数**

- 类的内置函数，**创建对象时自动执行**；
- 核心作用：给对象属性赋值、执行初始化操作。

1. **`self` 参数**

- 必须作为类方法**第一个参数**，指代当前类的实例；
- 用于访问实例的属性 / 方法，参数名可自定义（不强制叫 self）。

1. **对象方法**

   

   在类内部定义的函数，属于对象，调用方式：

   ```
   对象.方法名()
   ```

   。

2. **属性与对象操作**

- 直接赋值修改对象属性；
- 用 `del` 关键字：删除对象属性 / 删除整个对象。

1. **`pass` 语句**

   

   用于空类占位，解决类定义无内容时报错的语法问题。

## 1. 对象

- 通俗理解：**动作的操作目标**（教学生、看电视、开汽车 → 学生 / 电视 / 汽车 就是对象）
- 组成：**属性（特征） + 方法（行为）**
- 本质：**具体、真实存在的个体**，也叫**实例**

## 2. 类

- 本质：**抽象的模板 / 图纸**，只描述对象的通用特征，不对应具体事物
- 作用：统一规定一类对象的属性和方法

## 3. 核心关系

**类（抽象模板） → 实例化 → 对象（具体实例）**
