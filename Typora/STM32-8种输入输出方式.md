# STM32-8种输入输出方式

![image-20260401190947513](C:\Users\17210\AppData\Roaming\Typora\typora-user-images\image-20260401190947513.png)

TIPS: n-mos管低电平导通，高电平断开；p-mos管相反

推挽输出：推挽输出（Push-Pull Output）是数字集成电路、微控制器 GPIO 口、功率放大与驱动电路中最主流的输出级结构，核心由一对特性对称、互补配对的有源开关器件（主流为 PMOS+NMOS 场效应管，也可由 PNP+NPN 三极管搭建）串联在电源与地之间构成，两个器件受同一输入信号的反相控制，始终遵循 “一管导通、一管截止” 的互补工作原则，分别完成 “推” 与 “挽” 两个核心动作，实现高低电平的主动强驱动输出。

​        其完整工作过程可分为两个核心状态：当需要输出高电平时，上侧连接电源的 PMOS 管导通、下侧接地的 NMOS 管截止，电流从电源经导通的 PMOS 管直接流向负载，为负载提供拉电流，将输出电平快速上拉至接近电源电压，完成 “推” 的动作；当需要输出低电平时，NMOS 管导通、PMOS 管截止，负载电流从输出端经导通的 NMOS 管灌入地，将输出电平快速下拉至接近地电位，完成 “挽” 的动作。这种结构让推挽输出在高低电平状态下都具备极低的输出阻抗，无需外部上拉 / 下拉电阻即可输出完整的高低电平，兼具极强的灌电流与拉电流驱动能力，同时电平切换速度快、信号边沿陡峭、静态功耗极低，广泛应用于 MCU 通用 IO 口驱动小功率负载、SPI/UART 等点对点高速通信接口、功率放大电路输出级等场景；电路设计中必须保证两管切换时的死区时间，绝对避免上下管同时导通造成电源对地短路，烧毁器件。

开漏输出：开漏输出（Open-Drain Output，简称 OD 输出）是与推挽输出互补的经典数字输出结构，核心是完全移除了推挽输出级中连接电源的上侧 PMOS 管，仅保留接地的 N 沟道 MOS 管（NMOS），输出端仅从 NMOS 管的漏极引出，因漏极无内部电路连接、处于开路状态而得名；与之对应的双极型三极管结构为开集输出（Open-Collector，OC 输出）。其工作逻辑完全由这颗 NMOS 管的通断控制，仅能主动完成低电平输出，高电平必须依赖外部电路实现：当输入控制信号为高电平时，NMOS 管饱和导通，输出端（漏极）通过导通的 NMOS 管直接接地，输出稳定的低电平（接近 0V），具备较强的灌电流驱动能力；当输入控制信号为低电平时，NMOS 管完全截止，输出端进入高阻悬空状态，自身无法主动输出高电平，必须在输出端与目标电源之间外接一个上拉电阻，才能通过上拉电阻将输出电平拉升至对应电源的电压，实现高电平输出。

开漏输出最核心的优势在于电平适配灵活性与总线共享能力，其高电平电压完全由外部上拉电源决定，而非芯片自身的供电电压，可轻松实现不同电平域器件的跨电压通信（如 3.3V MCU 与 5V 外设的电平转换）；同时多个开漏输出引脚可并联在同一条总线上、共用一个上拉电阻，天然实现 “线与” 逻辑 —— 只要任意一个开漏输出导通拉低总线，整条总线即为低电平，仅当所有开漏输出均截止时，总线才被上拉为高电平，这一特性使其成为 I2C、SMBus、1-Wire 等多设备共享通信总线的标准输出结构。此外，开漏输出无推挽电路上下管同时导通的电源短路风险，可直接用于驱动 LED、继电器等需要大电流的负载，但其高电平驱动能力完全由上拉电阻决定，过大的电阻会引入 RC 延迟、限制信号切换速度，因此不适合高速点对点通信场景，上拉电阻通常需在几 kΩ 到几十 kΩ 之间选型，兼顾功耗、速度与驱动能力。

Tips：推晚输出时小灯是由来自芯片的电压所驱动，但开漏输出却并没有驱动能力，必须依靠外部的电压源来驱动，这是其缺点，但也因此而拥有了最大的优点，更加灵活了。开漏输出需要使用支持5伏容忍的io口，否则会使上方板块二极管长期导通，将5伏引入电源中造成损坏。

**普通推挽 / 普通开漏（通用 GPIO 模式）**

引脚的控制权完全归属 GPIO 外设模块，由 CPU 通过写 GPIO 的输出数据寄存器（ODR）、置位 / 复位寄存器（BSRR）直接控制引脚电平，引脚的高低电平状态完全由软件手动决定，和 MCU 内部其他片上外设完全无关。此时引脚的核心定位是通用输入输出口，用于基础的开关量控制、电平读写。

**复用推挽 / 复用开漏（外设复用模式）**

引脚通过内部多路选择器，将控制权从 GPIO 寄存器切换给 MCU 内部的片上外设（如 USART、SPI、I2C、定时器、CAN 等），此时 GPIO 的 ODR 寄存器完全失去对引脚的控制权，引脚的电平翻转、输出时序完全由对应外设的硬件自动驱动，软件仅需配置外设参数，无需手动操作 GPIO 寄存器。此时引脚的核心定位是片上外设的对外物理接口，用于实现专用的通信、定时、信号输出功能。

当我们设置上拉输入时，上拉电阻启用；设置下拉输入时，下拉电阻启用；而浮空输入时，两个电阻都不会启用。与开漏输出类似，可以依赖外部的电路实现更多可能性，电流继续前进，会遇见一个ttl肖特基触发器或者说施密特触发器，其作用是稳定电平，由于外部输入的电压可能不太稳定，并不总能达到标准的3.3伏或者0伏因而我们可以简单的规定一个参考电压，例如1.6伏高于此电压的就算作高电柄，低于此电压的就算做低电平，使用一个比较器就可以实现此效果
